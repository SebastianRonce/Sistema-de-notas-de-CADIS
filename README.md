# Sistema de Gestión de Calificaciones (CADIS)

Este proyecto en Python permite gestionar las calificaciones de un estudiante en diferentes materias (CADIS).  
El sistema incluye operaciones para **agregar, modificar, eliminar y calcular el promedio** de las notas.



# Funcionalidades

-Ingreso del nombre del estudiante.  
- Selección de materia (CADI).  
-Menú de operaciones:  
-Agregar nota  
-Modificar nota existente 
-Eliminar nota  
- Calcular promedio
- Validaciones en todas las entradas para evitar errores.  



# Materias disponibles (CADIS)

1. Cálculo Multivariado  
2. Física III  
3. Arquitectura de Computadores  
4. Fundamentos Administrativos  
5. Lengua Extranjera  
6. Estructuras de Información
